- ls ワーキングディレクトリの詳細確認可能 

  - option一覧
    - ls -a 不可視ファイルの表示
    - ls -l タイプ、権限、所有者など詳細を表示
    - ls /  全てのディレクトリやファイルを表示
    - ls /○○ 指定した〇の中のディレクトリが確認できる

- pwd ワーキングディレクトリを出力してくれる

- cd 指定したディレクトリに変更する
  - cd . ワーキングディレクトリの一階層上のディレクトリ
  - cd ../ ワーキングディレクトリの二階層上のディレクトリ

- && and演算子、左辺の実行が成功すれば、右辺を実行する

- cp fileやディレクトリのコピーをする
  - cp -r option「-r」を使用するとディレクトリのコピー

- mv file、ディレクトリを移動する

- rm fileやディレクトリを削除
  - rm -r option｢-r｣を使用するとディレクトリの削
    - rm -r f / fは警告メッセージなく削除される
    * /は最上位階位の/の配下が全て削除される

- cat 指定したファイルの内容をターミナル上に出力する

- > 左辺の実行結果を右辺で指定したファイルに書き込む
  * 既に存在するファイルだと上書きされる

- >> 左辺の実行出力結果を右辺で指定したファイルの末尾に書き込む
  * 存在しないファイルとの挙動は変わる

- vi ターミナル上で即時起動できるエディタコマンド

- grep ファイル中の文字列を検索する
